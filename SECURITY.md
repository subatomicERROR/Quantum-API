# Security Policy

The following table lists the supported versions of Quantum-API with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |
| < 1.0   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability in Quantum-API:

1. **Contact**: Send an email to yashramteke55555@gmail.com with details of the vulnerability.
2. **Provide Details**: Include steps to reproduce the issue, affected endpoints, and any relevant logs.
3. **Response Time**: We aim to acknowledge reports within 48 hours and provide updates regularly.
4. **Resolution**: If the vulnerability is accepted, a patch will be released in the next available update.

Thank you for helping keep Quantum-API secure!
